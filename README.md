# Cheerio Playground

A simple helper to evaluate Cheerio code in the browser. Inspired by [regex101](https://regex101.com).

Built with Svelte 5 and SvelteKit.

## Features

- Real-time HTML parsing with Cheerio
- Live code execution
- Keyboard shortcut (⌘/Ctrl + Enter) to run code
- Modern dark theme UI

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

1. Enter your HTML in the "HTML Input" textarea
2. Write your Cheerio selector/code in the "Cheerio Code" textarea
3. Click "Run" or press ⌘/Ctrl + Enter to execute
4. View the results in the "Output" panel
